Probleme
--------

o Inhalt des Warenkorbs nicht löschbar

  Wenn man ein Internetprodukt auswählt und auf "Bestellen" klickt,
  dann wird man zum Warenkorb weitergeleitet, aber man kann den Inhalt
  weder löschen bzw. irgendwo einsehen.

  Screenshot(s):  
  Video clip:     n03-GG-warenkorb.mp4

  Heuristik:      Umkehrbare Aktionen

  Wie reproduzierbar: Verzeichnisreiter → Internet → Internet Produkte → Bestellen


o Schwierigkeit den Speedtest zu finden

  Es gibt keinen direkten Link um einen Speedtest zu machen, sondern das funktioniert
  nur über Umwege. Erst später kommt man drauf, dass es einen eigenen Link im Kundenservice gibt.
  

  Screenshot(s):  n04-GG-speedtest-kundenservice.png
  Video clip:     n04-GG-speedtest.mp4

  Heuristik:      Erkennen ist besser als Erinnern

  Wie reproduzierbar: Verzeichnisreiter → Internet → Internet Produkte → Bestellen → Verzeichnisreiter → Speedtest


o Ein Link der auf die selbe Seite verweist

  Will man einen Speedtest machen, dann findet man auf der selben Seite einen Link, welcher wieder zur selben Seite
  (Speedtest) führt.

  Screenshot(s):  
  Video clip:     n36-GG-speedtest-link.mp4

  Heuristik:      Umkehrbare Aktionen

  Wie reproduzierbar: Verzeichnisreiter → Internet → Internet Produkte → Bestellen → Verzeichnisreiter → Speedtest → Link Speedtest


o Home Button anklicken funktioniert nicht

  Manchmal ist der Home Button im Verzeichnisreiter kleiner als sonst und es passiert nichts, wenn man ihn berührt.
  Berührt man aber die Fläche daneben, dann wird man auf die Startseite weitergeleitet.

  Screenshot(s):  
  Video clip:     n27-GG-home-button-bug.mp4

  Heuristik:      Fehlervermeidung

  Wie reproduzierbar: Verzeichnisreiter → Internet → Internet Produkte → Bestellen


o Desktop Ansicht wird angezeigt und nicht die mobile Ansicht

  Wenn man den Link "Digital Telefon" anklickt, dann kommt man automatisch auf die Desktop Ansicht.
  
  Screenshot(s):  n33-GG-desktopansicht.png
  Video clip:     

  Heuristik:      Fehlervermeidung

  Wie reproduzierbar: Verzeichnisreiter → Internet → Internet Produkte → Bestellen → Verzeichnisreiter → Digital Telefon


o Keine Möglichkeit Tarife zu vergleichen

  Will man einen Internet- oder Handytarif untereinander vergleichen, dann ist dies nicht möglich.
  
  Screenshot(s):  
  Video clip:     n07-GG-tarife-nicht-vergleichbar.mp4

  Heuristik:      Flexibilität und Efﬁzienz

  Wie reproduzierbar: Verzeichnisreiter → Internet → Internet Produkte → Produkte versuchen zu vergleichen

  Klickt man sich durch die unterschiedlichen Verzeichnisse im Verzeichnisreiter, dann sind mal mehr, mal weniger
  Optionen zur Auswahl möglich.
  
  Screenshot(s):  n15-GG-verzeichnisreiter-1.png
                  n15-GG-verzeichnisreiter-2.png
  Video clip:     

  Heuristik:      Konsistenz

  Wie reproduzierbar: Verzeichnisreiter → Internet → Internet Produkte → Bestellen → Verzeichnisreiter → Internet

  Bei "Sicherheit im Web" befindet sich ganz unten ein Link, der einen zu den Internetprodukten leiten sollte.
  Klickt man diesen jedoch an, dann landet man auf einer leeren Seite.
  
  Screenshot(s):  
  Video clip:     n01-GG-link-funktioniert-nicht.mp4

  Wie reproduzierbar: Verzeichnisreiter → Internet → Internet Produkte → Bestellen → Verzeichnisreiter → Internet → Sicherheit → Link       Internet
